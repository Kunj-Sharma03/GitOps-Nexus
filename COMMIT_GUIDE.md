# Quick Commit Commands

## Commit Types
- `feat:` - New feature
- `fix:` - Bug fix
- `docs:` - Documentation
- `style:` - Formatting, styling
- `refactor:` - Code restructuring
- `test:` - Adding tests
- `chore:` - Maintenance tasks

## Quick Usage
```bash
# Linux/Mac
./commit.sh "feat: add login endpoint"

# Windows (Git Bash)
bash commit.sh "feat: add login endpoint"

# Or manually
git add .
git commit -m "feat: add login endpoint"
git push
```
